Welcome to Lazy Loading Demo

This Angular app demonstrates lazy loading — each feature module is bundled into a separate chunk and only downloaded when the user navigates to that route. Open the Network tab and click the nav links to see it in action.

Username Availability Checker

Type a username — the API is called after you stop typing (500ms debounce). If you start typing again before the response arrives, the previous request is cancelled via switchMap. Check the Network tab to see it.

Chained API: User Search → Profile

Two sequential API calls chained with switchMap:
Step 1 GET /api/user/search?name=… — finds the user and returns their id.
Step 2 GET /api/user/:id/profile — uses that id to fetch the full profile.
Try: alice, bob, carol, deepak, eva
